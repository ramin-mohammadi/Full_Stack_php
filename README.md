# Full-Stack Equipment Inventory Management System (PHP)
Repository for assignemnts during advanced software engineering course taken for bachelors in CS at University of Texas at San Antonio.
- Developed RESTful API endpoints in PHP with comprehensive input validation, SQL injection prevention, and error 
handling/logging across 17 endpoints (add, modify, search, list devices/manufacturers/equipment)  
- Created dual-server infrastructure on AWS EC2 with Nginx, implementing dedicated API server and web server with cURL 
routing to centralize API logic while providing web interface for CRUD operations
